# mongoose-prototype
A playground for learning mongoose with MongoDB

## Getting start

### Local environment setup

1. Make sure you have installed the followings:

* [Node.js](https://nodejs.org/en/)
* [Yarn](https://yarnpkg.com/en/)
* [Docker](https://www.docker.com/)

2. Run

```bash
cd <project_root>;
yarn
yarn start
```
